# agar-plate-lightbox
An open-source, 3D-printable lightbox designed to support standardised smartphone-based imaging of agar plates. The device is intended to reduce glare, improve illumination consistency, and support downstream visual or computational analysis of microbial growth on agar plates.
This repository contains the design files, print-ready files, bill of materials, assembly instructions, and example documentation for the agar plate imaging lightbox.
