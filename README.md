# Agar Plate Lightbox

An open-source, 3D-printable lightbox designed to support standardised smartphone-based imaging of agar plates.

The device is intended to reduce glare, improve illumination consistency, and support downstream visual or computational analysis of microbial growth on agar plates.

## Repository contents

This repository contains the design files and print-ready files for the agar plate imaging lightbox.

| File                                       | Description                                                 |
| ------------------------------------------ | ----------------------------------------------------------- |
| `Micro Light Box - Box (iPhone) XL v1.f3d` | Editable Fusion 360 design file for the main lightbox body. |
| `Micro Light Box - Box (iPhone) XL.stl`    | Print-ready STL file for the main lightbox body.            |
| `Micro Light Box XL - Blocker Plate.stl`   | Print-ready STL file for the blocker plate.                 |
| `Micro Light Box XL - Diffuser Plate.stl`  | Print-ready STL file for the diffuser plate.                |
| `Micro Light Box XL - Insert.stl`          | Print-ready STL file for the agar plate insert.             |

## Suggested print settings

The following print settings were used or are suggested for standard desktop 3D printing:

* Material: PLA or PETG
* Layer height: 0.2 mm
* Infill: 15–20%
* Supports: As required depending on print orientation

Users may need to adjust these settings depending on their printer, filament and slicing software.

## Intended use

This lightbox was designed to provide a simple, low-cost and reproducible method for imaging agar plates using a smartphone camera. It may be useful for microbiology teaching, colony morphology documentation, haemolysis assessment, and downstream image analysis workflows.

## Assembly

1. Print each STL file.
2. Place the diffuser plate inside the lightbox body.
3. Insert the blocker plate and agar plate insert as required.
4. Position the agar plate within the insert.
5. Image the plate using a smartphone positioned above the lightbox.
